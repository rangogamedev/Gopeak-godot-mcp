**GoPeak 是一个面向 Godot 的 MCP 服务器，让 AI 助手可以对真实项目进行运行、检查、修改与调试。**

## 快速开始

### 要求
- Godot 4.x
- Node.js 18+
- 支持 MCP 的客户端

### 运行
```bash
npx -y gopeak
```

或者：
```bash
npm install -g gopeak
gopeak
```

## 核心概览
- 33 个核心工具 + 22 个按需激活的动态工具组
- 覆盖场景、脚本、资源、运行时、LSP、DAP 与输入自动化
- 适合基于真实 Godot 项目状态的 AI 工作流
