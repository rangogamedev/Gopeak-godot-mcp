Scope: worker-2 audit of MCP resources/prompts/dynamic exposure for the Godot 4.x-native correctness audit.
Repo snapshot: `0dfa4af` plus current working tree baseline.

## External MCP Evidence

- MCP resources are a server capability for exposing contextual data, with `resources/list`, `resources/read`, and `resources/templates/list`; custom URI schemes are allowed but must validate URIs and sanitize paths.
  - Source: <https://modelcontextprotocol.io/specification/draft/server/resources>
- MCP prompts are user-controlled prompt templates exposed by `prompts/list` and `prompts/get`; implementations should validate prompt arguments and support pagination for large prompt lists.
  - Source: <https://modelcontextprotocol.io/specification/draft/server/prompts>
- MCP pagination applies to list operations, including tools, prompts, resources, and resource templates.
  - Source: <https://modelcontextprotocol.io/specification/draft/server/utilities/pagination>
